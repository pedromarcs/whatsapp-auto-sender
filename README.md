## 🛠️ Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:
1. **Python 3.x**: [Download aqui](https://www.python.org/downloads/)
2. **Pip**: Gerenciador de pacotes do Python (geralmente vem com o Python).
3. **Navegador**: Google Chrome ou Edge (com login feito no WhatsApp Web).

---

## 🔧 Instalação

1. Clone este repositório para sua máquina local:
   ```bash
   git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
2. Instale as bibliotecas necessárias:
   ```bash
   pip install -r requirements.txt
3. Configure o script:
   ```bash
   Abra o arquivo main.py e altere as variáveis numero, mensagem e repeticoes.
4. Execute o programa:
   ```bash
   python main.py
